## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/rafaelsantos01/truco-lite.git
```

Entre no diretório do projeto

```bash
  cd truco-lite
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run dev
```

Api disponível em

```bash
  http://localhost:3000
```
